═══════════════════════════════════════════════════════════════
    📤 COMANDOS PARA SUBIR A GITHUB
═══════════════════════════════════════════════════════════════

IMPORTANTE: Reemplaza "TU-USUARIO" y "TU-REPO" con tus datos reales

═══════════════════════════════════════════════════════════════

PASO 1: Conectar con tu repositorio
────────────────────────────────────────────────────────────────

git remote add origin https://github.com/TU-USUARIO/TU-REPO.git


PASO 2: Subir los archivos
────────────────────────────────────────────────────────────────

git push -u origin main


═══════════════════════════════════════════════════════════════

EJEMPLO (si tu usuario es "juan" y tu repo es "burgerpoint"):
────────────────────────────────────────────────────────────────

git remote add origin https://github.com/juan/burgerpoint.git
git push -u origin main


═══════════════════════════════════════════════════════════════

DESPUÉS DE SUBIR:
────────────────────────────────────────────────────────────────

1. Ve a tu repositorio en GitHub
2. Click en "Settings" (arriba)
3. Click en "Pages" (menú izquierdo)
4. En "Source" selecciona "Deploy from a branch"
5. En "Branch" selecciona "main"
6. Click en "Save"
7. Espera 2-3 minutos


TU SITIO ESTARÁ EN:
────────────────────────────────────────────────────────────────

https://TU-USUARIO.github.io/TU-REPO/burger.html


═══════════════════════════════════════════════════════════════

SI TE PIDE USUARIO Y CONTRASEÑA:
────────────────────────────────────────────────────────────────

- Usuario: Tu nombre de usuario de GitHub
- Contraseña: Usa un Personal Access Token (no tu contraseña)

Para crear un token:
1. GitHub → Settings → Developer settings → Personal access tokens
2. Generate new token (classic)
3. Marca "repo"
4. Copia el token y úsalo como contraseña
